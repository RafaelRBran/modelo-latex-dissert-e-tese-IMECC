Este modelo está seguindo as normas preconizadas pela Informação CCPG nº 002/2021.

Para usar este modelo de trabalho acadêmico, você deve estar utilizando codificação UTF8 no seu editor LaTeX.
Esta configuração se aplica a todos os arquivos envolvidos à produção do trabalho acadêmico usando este
modelo.

Recomendo que leia todo o documento PDF gerado 'trabalho.pdf' antes de realizar quaisquer modificações
no código fonte deste modelo. Há muito aprendizado, dúvidas e sugestões que podem resolver seus problemas.

Este modelo é apenas uma adaptação do Modelo Canônico de Trabalho Acadêmico fornecido pela equipe do 
abnTeX2 em http://www.abntex.net.br/. Portanto, para melhor entendimento da dinâmica deste modelo, tenha
em mãos os manuais da classe abntex2, que podem ser obtidos no próprio site.

O arquivo 'imecc-unicamp.cls' é essencial para este modelo E NÃO DEVE SER ALTERADO/REMOVIDO/SUBSTITUÍDO.

O arquivo 'trabalho.tex' é o arquivo principal. É ele que você deve compilar, por isso altere apenas o
essencial nele. Configure seu editor para que o arquivo principal de compilação seja 'trabalho.tex'.

O arquivo 'pacotes.tex' contém os pacotes que estão sendo usados atualmente pelo modelo. Perceba que nele
estão alguns pacotes essenciais para o modelo, os quais não devem ser removidos. Leia o arquivo e faça suas
alterações desejadas.

O arquivo 'configglobal.tex' fornece o conjunto de configurações gerais para todo o documento. Caso você
esteja usando pacotes que requerem um conjunto de definições e configurações específicas, coloque-as
neste aquivo. Leia-o e faça suas alterações desejadas.

O arquivo 'dados.tex' contém todas as informações necessárias para a impressão da capa e da folha de rosto
de seu trabalho acadêmico.

Os arquivos 'anexos.tex', 'apendices.tex', 'introducao.tex', 'considfinais.tex', 'resumos.tex',
'abntex2-modelo-exemplos.tex', 'abntex2-modelo-img-grafico.pdf' e 'abntex2-modelo-img-marca.pdf'
são arquivos cujo conteúdo está presente apenas como exemplo. Destes, apenas os arquivos 'introducao.tex',
'resumos.tex' e 'considfinais.tex' não podem ser descartados (apenas alterados). Os demais podem ser 
descartados/alterados/substituídos.

O arquivo 'abntex2-modelo-references.bib' pode ser substituído pelo arquivo .BIB que contém as referências
do seu trabalho acadêmico. Perceba que este modelo necessita de um arquivo .BIB contendo bibliografias.
Utilize um gerenciador de bibliografias (JabRef, Mendeley, Zotero ou outro equivalente). Note que todos os
sites de revistas científicas possibilitam o download do arquivo .BIB correspondente àquele artigo. NÃO
CONFIE 100% NO ARQUIVO .BIB DISPONIBILIZADO PELO SCHOLAR GOOGLE.
